## Why This Matters

Using consistent, precise language in code (class names, method names, variable names,
documentation) reduces cognitive load and prevents bugs caused by misunderstanding.
When an AI tool knows that a "SKU" is a stock-keeping unit distinct from a "Product",
it will not conflate the two in generated code.
